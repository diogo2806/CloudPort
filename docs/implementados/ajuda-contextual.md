# Ajuda contextual

Implementação concluída no portal React:

- botão de ajuda adicionado ao componente compartilhado `PageHeader`;
- cobertura automática para páginas atuais e futuras que usem o cabeçalho padrão;
- catálogo por módulo e rota;
- finalidade, fluxo, campos, permissões, estados, bloqueios, exemplos e atalhos;
- pesquisa textual sem diferenciação de acentos;
- exibição dos papéis do usuário autenticado;
- navegação para o processo relacionado;
- acesso à documentação técnica;
- painel responsivo e acessível;
- atalhos `F1`, `Shift + ?` e `Esc`.
